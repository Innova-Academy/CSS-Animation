Animation Css

The animation Property:

(animation: mymove 5s infinite) make the animation lasts for 5 seconds. It then starts over again, and go on forever (infinite), @keyframes mymove { from {left: 0px;} to {left: 200px;} } : Move an element from 0px to 200px left. .

The animation-delay Property

(animation-delay: 3s) Start the animation after 3 seconds:

The animation-duration Property

( animation-duration: 3s)Specify that the animation should complete one cycle in 3 seconds:

The animation-fill-mode Property

>(animation-fill-mode: forwards)Let the div element retain the style values from the last keyframe when the animation ends:

The animation-iteration-count Property

(animation-iteration-count: 2)Play the animation two times:
